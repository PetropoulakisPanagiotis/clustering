## Clustering
<p align="left">
<img src="https://cssanalytics.files.wordpress.com/2013/11/cluster-image.png" width="480px" height="235px"> <br /> <br />

Available methods: 
* Initialization:
** random
** k-means++
* Assign: 
** lloyd's
** range(lsh/hypercube)
* Update: 
** k-means
** pam like lloyd's <br />

Metrices: euclidean and cosine

## Requirements
* Compiler: g++ - C11 standard

## Installation
* Clone this repo to your local machine:
