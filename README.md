## Available methods:
<strong>Initialization:</strong><br />
* random
* k-means++<br />

<strong>Assign: </strong><br />
* lloyd's
* range(lsh/hypercube)<br />

<strong>Update:</strong><br />
* k-means
* pam like lloyd's <br />

Metrices: euclidean and cosine
